# Dream-Surge

Currently this is a short breezy walk through a forest area; with emphasis on the visuals; though I would like for this to be a much bigger project in the future. I also admit my programming is a real mess at the moment which gave me more incentive to post it here so I may reminisce on its primal chaos in the future!

Controls (GBA):
Left/Right to Move, A to Jump, B to Wall Climb, L/R to change brightness
